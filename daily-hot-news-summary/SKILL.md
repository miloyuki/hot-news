# 今日热门资讯汇总技能

## 触发规则
当用户的请求符合以下任意场景时，必须调用本技能：
1.  用户询问今日/当日热门资讯、热点新闻汇总、每日新闻总结
2.  用户需要各领域最新热点、新闻简报、资讯MD文档
3.  用户明确要求获取当日新闻并生成可保存的Markdown文件

## 执行指令
1.  调用`run_js`工具，参数严格遵循以下规范：
    - script name: index.html
    - data: 字符串化的JSON对象，字段定义如下：
      | 字段名 | 类型 | 是否必填 | 说明 |
      |--------|------|----------|------|
      | newsCount | Number | 否 | 每个领域获取的资讯条数，默认5条 |
      | language | String | 否 | 资讯语言，默认zh（中文） |
      | country | String | 否 | 资讯地区，默认cn（中国） |

2.  接收JS脚本返回结果后，按以下规则处理：
    - 若返回`error`字段：直接向用户清晰说明错误原因，引导用户排查API密钥、网络等问题
    - 若返回`result`字段：严格按照下方【Markdown输出规范】，将资讯数据总结生成完整的Markdown文档

## Markdown输出规范
必须严格遵循以下格式生成文档，确保结构清晰、可直接保存为.md文件：
```markdown
# 【YYYY年MM月DD日】全网热门资讯汇总
> 文档生成时间：YYYY-MM-DD HH:mm
> 覆盖领域：科技、财经、文娱、体育、国际
